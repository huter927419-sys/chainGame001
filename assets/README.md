# 图片资源说明

请将以下F1赛车图片放置在此目录中：

## 需要的图片文件

1. **car-red.png** - 红色F1赛车
   - 描述：红色F1赛车，黑色背景
   - 尺寸建议：300x300px 或更大
   - 格式：PNG（支持透明背景）

2. **car-neon.png** - 紫色/霓虹色F1赛车
   - 描述：紫色/深蓝色F1赛车，带有霓虹粉色和蓝色发光线条
   - 尺寸建议：300x300px 或更大
   - 格式：PNG（支持透明背景）

## 图片用途

- **car-red.png**: 默认显示，加速道具效果时使用
- **car-neon.png**: 减速道具效果时使用

## 替代方案

如果暂时没有图片，可以使用：
- 在线图标库（如 Font Awesome 的赛车图标）
- SVG 矢量图
- 使用 CSS 绘制的简单赛车图形

## 图片优化建议

- 使用 WebP 格式以获得更好的压缩比
- 确保图片大小不超过 500KB
- 建议使用透明背景以便更好地融入游戏界面
